# E-COMMERCE-API
An example API using React,Express,MongoDB,Mongoose,Cloudinary,formidable, Swagger,winston,Zod,Typescript
# Setup 
Create a .env file and add the properties with values
MONGO_URI,
CLOUD_NAME,
CLOUDINARY_KEY,
CLOUDINARY_SECRET
PORT

# API end points
API endpoint to use: 
http://{host}:{port}/api/users
http://{host}:{port}/api/products
http://{host}:{port}/api/orders
http://{host}:{port}/api/categories
http://{host}:{port}/api/shoppingCarts
